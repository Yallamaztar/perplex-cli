PerpleX Sniper SnD Status Checker
=================================

A small command-line tool that shows the status of PerpleX Sniper SnD servers

![status](assets/image.png)

Quick start
-----------

Build (Windows, recommended):

```powershell
.\build.ps1
```

Run once:

```shell
status.exe
```

Run in a loop (updates every few seconds):

```shell
status.exe -loop
```
